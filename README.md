# EliteBot Discord documentation

# Commands
|      Command      	|                    Args                    	|                                 Description                                 	|                                                          	|
|:-----------------:	|:------------------------------------------:	|:---------------------------------------------------------------------------:	|----------------------------------------------------------	|
| e.ascii           	| <text>                                     	| Makes an ascii art for you.                                                 	| Notes                                                    	|
| e.mkembed         	| <title> <br><content>                      	| Makes an embed by user input.                                               	|                                                          	|
| e.editembed       	| <msg-id> <br><new title> <br><new content> 	| Edits an embed which is <br>already made by this bot.                       	| Not returning <br>errors except<br>syntax error.         	|
| e.ping            	| No args                                    	| Check it for yourself.                                                      	| First command.                                           	|
| e.checkstock      	| No args                                    	| Checks current stock values.                                                	| One time <br>message.                                    	|
| e.setstockchannel 	| No args                                    	| Sets the channel the command<br>was executed in as the <br>"stock" channel. 	| Updates every<br>5 minutes.                              	|
| e.remstockmsg     	| No args                                    	| Removes the current stock<br>message.                                       	|                                                          	|
| e.s-accept        	| <msg-id>                                   	| Accepts a suggestion based <br>on message id.                               	| Everyone will<br>know who <br>accepted it.               	|
| e.s-deny          	| <msg-id>                                   	| Denies a suggestion based<br>on message id.                                 	| Everyone will<br>know who<br>denied it.                  	|
| /status           	| <server-status>                            	| Sets the server status.                                                     	| First arg <br>accepts;<br>online<br>offline<br>debugging 	|
