# EliteBot Discord documentation

# Commands

|      Command      	|                Args                	|                                 Description                                 	|
|:-----------------:	|:----------------------------------:	|:---------------------------------------------------------------------------:	|
| e.ascii           	| <text>                             	| Makes an ascii art for you.                                                 	|
| e.mkembed         	| <title> <content>                  	| Makes an embed by user input.                                               	|
| e.editembed       	| <msg-id> <new title> <new content> 	| Edits an embed which is <br>already made by this bot.                       	|
| e.ping            	| No args                            	| Check it for yourself.                                                      	|
| e.checkstock      	| No args                            	| Checks current stock values.                                                	|
| e.setstockchannel 	| No args                            	| Sets the channel the command<br>was executed in as the <br>"stock" channel. 	|
| e.remstockmsg     	| No args                            	| Removes the current stock<br>message.                                       	|
| e.s-accept        	| <msg-id>                           	| Accepts a suggestion based <br>on message id.                               	|
| e.s-deny          	| <msg-id>                           	| Denies a suggestion based<br>on message id.                                 	|
| /status           	| <server status>                    	| Sets the server status.                                                     	|
